## Login

**로그인 및 계정 관련 컴포넌트**

```
└─ login (directory)
 ├─ findAccount (directory)   FindAccount의 하위 컴포넌트들로 이루어진 폴더
   └─ FindAccount             비밀번호 찾기 컴포넌트
 ├─ loginForm (directory)     LoginContainer의 하위 컴포넌트들로 이루어진 폴더
   ├─ LoginForm               로그인 정보 입력 컴포넌트
   ├─ LoginButton             로그인 버튼 컴포넌트
   └─ LoginError              로그인 에러 발생 컴포넌트
 ├─ Index                     시작 컴포넌트
 ├─ LoginContainer            로컬 로그인 기능 수행을 위한 컴포넌트
 ├─ LoginSocial               소셜 로그인 기능 수행을 위한 컴포넌트
 ├─ LoginTitle                로그인 로고 컴포넌트
 └─ Account                   로그인 외 계정 관련 컴포넌트
```
