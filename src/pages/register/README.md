## Register

**회원가입 관련 컴포넌트**

```
└─ pages/register (directory)
 ├─ registerForm (directory)  RegisterContainer의 하위 컴포넌트들로 이루어진 폴더
   ├─ RegisterButton          회원가입 버튼 컴포넌트
   ├─ RegisterEmailAuth       회원가입 이메일 인증 컴포넌트
   └─ RegisterForm            회원가입 정보 입력 컴포넌트
 ├─ Index                     시작 컴포넌트
 ├─ RegisterContainer         회원가입 기능 수행을 위한 컴포넌트
 ├─ RegisterTitle             회원가입창 로고 컴포넌트
 └─ RegisterTOC               회원가입 필요 정보를 보여주는 컴포넌트
```
